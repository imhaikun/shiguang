# 产品需求文档（PRD）- 拾光笔记博客

## 1. 产品概述
拾光笔记是一款以"编辑/温暖文学/衬线优先"为美学基调的个人博客站点，面向喜爱阅读与写作的开发者与文字爱好者。
- 解决问题：为独立创作者提供一个兼具阅读沉浸感与精致排版的内容发布与浏览空间。
- 目标用户：开发者、设计师、文字创作者；产品价值在于用克制的暖色调与衬线排版传递"慢阅读"体验。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览首页、文章详情、归档、关于，搜索与标签筛选 |
| 作者（隐含） | 后台预留 | 内容由 Mock 数据模拟，未来可扩展为 CMS 录入 |

### 2.2 功能模块
1. **首页**：站点顶栏、置顶精选文章、文章列表（含日期/标题/摘要/阅读链接）、右侧栏（作者卡、搜索、侧边导航、标签云）
2. **文章详情页**：文章标题、元信息、正文排版、标签、上下篇导航、返回首页
3. **归档页**：按月份分组的文章时间线
4. **关于页**：作者介绍、写作理念、联系方式

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 顶栏导航 | 站点标题"拾光笔记"，导航项首页/归档/关于，主题切换按钮 |
| 首页 | 置顶精选 | 赤陶土左边框强调卡片，含日期、大标题、摘要、阅读全文链接 |
| 首页 | 文章列表 | 日期+标题+单行摘要+阅读链接的横向条目，分隔线分隔 |
| 首页 | 作者卡 | 渐变头像、笔名"林屿"、签名"写代码，也写生活" |
| 首页 | 搜索框 | 关键词输入，聚焦时边框变为强调色 |
| 首页 | 侧边导航 | 最新文章/分类/标签云入口，当前项高亮 |
| 首页 | 标签云 | 可点击标签，hover 高亮，点击进入筛选结果 |
| 文章详情 | 文章头部 | 返回链接、日期、标题、标签 |
| 文章详情 | 正文排版 | 衬线标题层级、段落、引用块、代码块、列表 |
| 文章详情 | 上下篇导航 | 上一篇/下一篇卡片，无则禁用 |
| 归档页 | 时间线 | 按年月分组的文章列表，点击进入详情 |
| 关于页 | 作者介绍 | 头像、介绍文字、技能/兴趣标签、联系方式 |

## 3. 核心流程
访客进入首页浏览精选与文章列表 → 点击"阅读全文"进入文章详情阅读正文 → 通过上下篇或返回链接继续浏览；亦可通过搜索框或标签云筛选文章，或进入归档页按时间线查找。

## 4. 用户界面设计

### 4.1 设计风格
- 主色：暖石墨墨色 #2c2c2c（前景）+ 赤陶土 #c2703e（强调色）；背景暖石白 #faf8f5
- 次要色：muted #a8a29e / muted-foreground #57534e；边框 #e7e5e4
- 暗色模式：背景 #1a1917，卡片 #262523，强调色 #d4885a
- 字体：标题衬线 Playfair Display + Noto Serif SC；正文无衬线 Inter + Noto Sans SC
- 圆角：sm 4px / md 8px / lg 12px
- 布局：双栏编辑式（内容 1fr + 侧边 240px），最大宽度 1152px
- 图标：Lucide 线性图标，克制使用
- 动效：边框/颜色过渡 0.2s，链接箭头位移，页面渐入

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 首页 | 置顶精选 | 卡片白底、赤陶土左边框 3px、衬线大标题、日历图标日期 |
| 首页 | 文章列表条目 | 左侧固定宽度日期、中间标题+单行省略摘要、右侧阅读链接 |
| 首页 | 侧边栏 | 居中作者卡、搜索输入、导航列表、圆角小标签 |
| 文章详情 | 正文 | 衬线标题、1.8 行高段落、左边框引用、等宽代码块 |
| 归档页 | 时间线 | 年月分组标题 + 文章条目 + 分隔线 |
| 关于页 | 介绍 | 头像+介绍段落+标签+联系方式链接 |

### 4.3 响应式
- 桌面优先（≥1024px）：双栏 1fr + 240px
- 平板（768-1023px）：保持双栏但缩小间距
- 移动端（<768px）：单栏堆叠，侧边栏移至内容下方；导航简化为汉堡菜单或紧凑横排；触摸目标≥44px

### 4.4 3D 场景指引
不适用（本项目为纯 2D 排版型博客）。
